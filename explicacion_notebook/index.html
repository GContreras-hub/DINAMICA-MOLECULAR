    Explicación del Notebook

    Este archivo Jupyter Notebook contiene el desarrollo principal del proyecto de simulación de dinámica molecular con discos sólidos. Aquí se explican las secciones del notebook y cómo usarlo correctamente.

    Estructura general del notebook

    1. Importación de librerías

    2. Se importan numpy, matplotlib.pyplot, y los módulos creados en src/.

    3. Definición de parámetros del sistema

    4. L: tamaño del cubo simulado
    5. dt: paso de tiempo
    6. t: tiempo total de simulación

    7. N: cantidad de partículas

    8. Generación de partículas

    9. Se usa generar_posiciones(N) desde sim.py para inicializar partículas con posiciones, radios, masas y velocidades aleatorias.

    10. Se asegura que no haya solapamientos iniciales.

    11. Ejecución de la simulación

    12. Se llama a simular(particulas, t, dt, L) desde simulador.py, que ejecuta el bucle temporal completo.

    13. Se recolectan todas las posiciones en un array para análisis posterior.

    14. Visualización

    15. Se genera un histograma 3D de las posiciones acumuladas (np.histogramdd).
    16. Se grafican cortes planos del histograma: XY, XZ, YZ usando matplotlib.pyplot.imshow.

    Guía rápida para ejecutar el proyecto

    Este proyecto está diseñado para correr desde Jupyter Notebook. A continuación se resumen los pasos necesarios para ejecutarlo correctamente.

    1. Clonar el repositorio

    bash: git clone https://github.com/GContreras-hub/DINAMICA-MOLECULAR.git cd DINAMICA-MOLECULAR

    2. Instalar dependencias (de ser necesario)

    bash: pip install numpy matplotlib

    3. Abrir el notebook

    cd notebook jupyter notebook

    4. Abrir el archivo del proyecto

    proyecto_final.ipynb

    5. Ejecutar el notebook

    • Ejecutá todas las celdas en orden, usando Shift + Enter.
    • Al finalizar, se mostrarán los gráficos generados por la simulación.

    6. Importar desde src/ (de ser necesario)

    import sys sys.path.append('../src')
